# User Guide

This short guide shows common user flows in the e-commerce platform.

1) Browse products
   - Open the homepage at `/`.
   - Click `Products` to view a list.
   - Use search and filters to narrow results.

2) View product details
   - Click a product to open `/products/:id`.
   - Choose options (size, color) if available and click `Add to Cart`.

3) Register and login
   - Register at `/register` with name, email, and password.
   - Login at `/login` to access profile and checkout features.

4) Manage cart
   - Visit `/cart` to review items.
   - Update quantities or remove items.
   - Proceed to `/checkout` to enter shipping and payment details.

5) Place an order
   - On checkout, confirm shipping and payment.
   - After placing an order, you will receive order confirmation and (if enabled) real-time order updates.

6) Profile
   - Visit `/profile` to view or edit account details and addresses.

Tips
- Keep the browser open to receive real-time stock and order notifications.
- For security, log out when using shared devices.
